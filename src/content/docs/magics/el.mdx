`$el` — это магическое свойство, которое может быть использовано для получения текущего узла DOM.

```html live "$el"
<div x-data>
  <button x-on:click="$el.innerHTML = 'Привет, мир!'">
    Замени меня на «Привет, мир!»
  </button>
</div>
```
