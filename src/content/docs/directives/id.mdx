`x-id` позволяет вам объявить новую «область действия» для любых новых идентификаторов, созданных с помощью `$id()`. Он принимает массив строк (имена идентификаторов) и добавляет к каждому сгенерированному в нём суффиксу `$id('...')`, который уникален для других идентификаторов на странице.

`x-id` предназначен для использования в сочетании с магией `$id(...)`.

<LinkCard title="Подробнее об <strong>$id</strong>" href="/magics/id" />

Вот краткий пример использования этой директивы:

```html "x-id"
<div x-id="['text-input']">
  <label :for="$id('text-input')">Имя пользователя</label>
  <!-- for="text-input-1" -->

  <input type="text" :id="$id('text-input')" />
  <!-- id="text-input-1" -->
</div>

<div x-id="['text-input']">
  <label :for="$id('text-input')">Имя пользователя</label>
  <!-- for="text-input-2" -->

  <input type="text" :id="$id('text-input')" />
  <!-- id="text-input-2" -->
</div>
```
